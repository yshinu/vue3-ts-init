# Vite4.3+Typescript+Vue3+Pinia 最新搭建企业级前端项目
## 项目采用pnpm包管理器配置了以下内容：
- ESLint
- Prettier
- Husky
- 环境变量
- Pinia
- Vue Router
- SCSS
- 封装 Axios
- postcss-pxtorem
- 简单的性能优化



## 代码提交规范
| 关键词 | 描述 |
| --- | --- |
| `feat` | 新功能 |
| `fix` | 修复 bug |
| `style` | 样式修改（UI 校验） |
| `docs` | 文档更新 |
| `refactor` | 重构代码（既没有新增功能，也没有修复 bug） |
| `perf` | 优化相关，比如提升性能、体验 |
| `test` | 增加测试，包括单元测试、集成测试等 |
| `build` | 构建系统或外部依赖项的更改 |
| `ci` | 自动化流程配置或脚本修改 |
| `revert` | 回退某个 commit 提交 |
